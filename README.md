### Описание

В данном коде собран готовый инструмент для воспроизведения видео. Смело используйте его на своём сайте после добавления файлов и кода в html.

### Как пользоваться

Чтобы воспроизвести ваше видео, добавьте ссылку на него в файл player.js:
```src='https://dvmn.org/media/filer_public/78/db/78db3456-3fd3-4504-9ed9-d2d1fd843c0b/highest_peak.mp4```

### Цель проекта

Код написан в образовательных целях на онлайн-курсе для веб-разработчиков [dvmn.org](https://dvmn.org/).
